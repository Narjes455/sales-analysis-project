# 📊 مشروع تحليل بيانات المبيعات

هذا المشروع يهدف إلى تحليل بيانات مبيعات متجر إلكتروني باستخدام Python، وإخراج تقارير ورسوم بيانية تساعد في فهم أداء المنتجات والمناطق.

---

## 🛠️ التقنيات المستخدمة

- **Python 3**
- **Pandas** – معالجة البيانات
- **Matplotlib** – الرسوم البيانية
- **Seaborn** – رسوم بيانية احترافية
- **OpenPyXL** – التعامل مع Excel

---

## 📁 محتويات المشروع

| الملف | الوصف |
|-------|-------|
| `sales_analysis.py` | الكود الأساسي لتحليل البيانات |
| `sales_data.xlsx` | بيانات المبيعات الوهمية |
| `sales_by_product.png` | رسم بياني للمبيعات حسب المنتج |
| `sales_by_region.png` | رسم بياني للمبيعات حسب المنطقة |

---

## ▶️ طريقة التشغيل

1. تأكد من تثبيت المكتبات المطلوبة:
```bash
pip install pandas numpy matplotlib openpyxl seaborn
```

2. شغّل الكود:
```bash
python sales_analysis.py
```

3. ستظهر النتائج في ملفات جديدة داخل المجلد.
